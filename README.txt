Setting up Python, Pygame

	Step 1: Install python-2.7.10.amd64.msi 
	Step 2: Copy pygame-1.9.2-cp27-cp27m-win_amd64.whl in C:\Python27\Scripts\
	Step 3: Open Command Prompt in Administrator mode and go to C:\Python27\Scripts\
	Step 4: Upgrade pip , Type cmd " pip install --upgrade pip "
	Step 5: For Install Pygame , Type command " pip install pygame-1.9.2-cp27-cp27m-win_amd64.whl "
	Step 6: To confirm Installation Open cmd -> type "python" hit enter -> 

Below message should appear after successfull installation.

Python 2.7.10 (default, May 23 2015, 09:44:00) [MSC v.1500 64 bit (AMD64)] on win32
Type "help", "copyright", "credits" or "license" for more information.
>>>

	Step 7: Type "import pygame" hit enter ->

Below message should appear after successfull installation.

Python 2.7.10 (default, May 23 2015, 09:44:00) [MSC v.1500 64 bit (AMD64)] on win32
Type "help", "copyright", "credits" or "license" for more information.
>>> import pygame
>>>

If no error , your system is ready to play Carrom.

Steps to run the game

Method 1: From Command Prompt
			Step 1: Open Command Prompt and go to Main.py , carrom.py file location directory.
			Step 2: Type " python Main.py " hit enter to start the game.
			
Method 2: From Visual Studio
			Step 1: Install Python Tools For Visual Studio(PTVS).
			Step 2: Open the CarromGame.sln from folder CarromGame.
			Step 3: Click Start with Main.py as startup file to run the game.
